CODE FUSION

Event Description:
IEEE Computer Society Student Branch Chapter SBC34451D is organizing CODE FUSION Event. You can claim One Credit in Experiential Elective by participating in this event. This event is designed to prepare students for the IEEE Xtreme, an International programming competition, by enhancing their coding, problem-solving, and teamwork skills.

Organized in two rounds from 9:30 AM to 5:00 PM, this event simulates the competitive environment of IEEE Xtreme, allowing participants to practice tackling complex coding challenges under time constraints.

Coding Languages: C/Java
Coding Platform: HackerRank

The event aims to boost technical proficiency, collaboration, and readiness for global coding competitions, aligned with Sustainable Development Goal (SDG) 4: Quality Education.
